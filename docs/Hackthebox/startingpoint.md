# starting point machines

## Meow

```shell
sudo namp -sN -A <ip_address> -v
telnet <ip_address>
username: root
cat flag.txt
```

## Fawn

```shell
sudo nmap -sN -A <ip_address> -v
ftp <ip_address>
username: anonymous
get flag.txt
exit
cat flag.txt
```
